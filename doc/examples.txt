==========
 Examples
==========

Refer to ``examples/src`` for a list of examples provided with Landshark. Many
have already been covered in the documentation, but there are some more
advanced examples as well.

To run an example:

.. code-block:: bash

   cd ``examples/src``
   make MOD=<module> run

where ``<module>`` is the module name you want to run.

For example, to run the ``userlogin`` example, run ``make MOD=userlogin run``.
